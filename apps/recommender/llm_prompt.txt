На основе этой информации определи, содержит ли она в себе информацию об IT-проекте. Если да, выведи результат, согласно инструкции ниже. Если нет, ответь просто False, одним словом.

Инструкция
Из данного описания извлеки информацию следующего характера:
- Вид проекта (веб-приложение, десктопное приложение или мобильное приложение - только эти)
- Планируется ли хранить данные в реализуемом сервисе (да/нет)
- Будет ли система работать с большими данными (да/нет)
- Структура данных, если хранятся (выбор: SQL или NoSQL)
- Будет ли проводиться анализ данных (да/нет)
- Планируется ли использовать машинное обучение (да/нет)
- Технологии должны быть open source или проприетарные (по умолчанию: open source)
- Планируется ли масштабирование проекта в будущем? (да/нет)
- Планируется ли автоматическое тестирование проекта? (да/нет)

Ответы на данные вопросы дай полностью на русском языке, выбери из предложенных в скобках и представь в формате JSON, следующего вида:
{
"product_type": String,
"db_needed": Boolean,
"big_data_needed": Boolean,
"data_structure": String,
"data_analysis_needed": Boolean,
"ml_needed": Boolean,
"licensing_type": String,
"scaling_needed": Boolean,
"autotesting_needed": Boolean,
}
В своем ответе дай только JSON, без дополнительных символов. Либо слово False, если пользовательский запрос не соотвествует тематике.
